# Java + React Full Stack Engineering – 7 Week Training Journey

## Overview

This repository contains all the assignments, hands-on labs, and projects completed during the **7-week Java + React Full Stack Engineering Training Program**. The program focused on building end-to-end development skills, starting from software engineering fundamentals to modern full-stack application development using Java, Spring Boot, React, Git, DevOps, and Cloud concepts.

Throughout the training, I worked on multiple practical exercises to strengthen my understanding of backend development, frontend development, database management, software testing, version control, and deployment practices.
